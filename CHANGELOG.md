## [0.4.0] - 2025-01-07

### Added
- Media file conversion to MP3 with batch processing
- YouTube and other platform audio/video downloading via yt-dlp
- Telegram bot integration for remote download delivery
- Queue management with pause/resume/cancel controls
- Settings persistence for theme, output folder, and quality preferences
- Automatic dependency checking and installation
- Auto-update functionality via electron-updater
- Temporary file cleanup on startup and manual trigger
